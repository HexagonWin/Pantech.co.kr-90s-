Security systems, using Keys, ID cards and passwords have fundamental shortcomings : they are vulnerable to copying or unauthorized use. If they are lost or exposed to other parties, it creates not only inconvenience but also problems which might lead to serious damage.

The Pantech Media's fingerprint recognition system is an advanced biometric security system which has completely solved problems inherent to the systems currently in use. Fingerprints are unique to each individual. If they are registered, the system later identifies them to determine if users are authorized persons.

Now, you can get rid of all heavy keys, ID cards and the easily exposed passwords.  Instead, you are protected by an advanced security system which identifies "you" by means of simple movement of your fingertip.

FINKEY

 

FINKEY is a system which opens doors (with locking devices) through the use of fingerprints. The doors are opened only when the pre-registered fingerprints exactly match those of the persons desiring to enter.
As the FINKEY system is based on the uniqueness of fingerprints, security is totally guaranteed. Keys or  ID cards are easily lost or stolen, but your fingerprints are always at your disposal. Finkey also eliminates the inconvenience of having to carry keys or ID cards at all times.

FINSIGN

 

 FINSIGN fingerprint recognition system can be connected to any PC without using any separate processing system. It is composed of a fingerprint scanner, an image processing board and an advanced fingerprint recognition algorithm software.
This fingerprint recognition algorithm is provided in the form of a library. It can be used for independent operations, such as control of PC use or data security, and it can alse be used in the existing officeware or in-house computer system environment.
